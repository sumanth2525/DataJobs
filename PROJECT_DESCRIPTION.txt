================================================================================
                      DATA JOB PORTAL - PROJECT DESCRIPTION
================================================================================

PROJECT NAME: DataJobPortal (Jobright)
VERSION: 1.0.0
DESCRIPTION: A modern, responsive job portal platform specialized for data-related roles

================================================================================
                              PROJECT OVERVIEW
================================================================================

DataJobPortal is a comprehensive web application designed to help job seekers 
find data-related positions including Data Analyst, Data Scientist, Data Engineer,
ETL Developer, Business Intelligence Analyst, and other data-focused roles. The
platform features a clean, modern design with a three-column layout, AI-powered
job recommendations, and seamless integration with external job APIs.

================================================================================
                            KEY FEATURES
================================================================================

1. JOB LISTINGS & SEARCH
   - Browse and search data-related job postings
   - Advanced filtering by location, experience level, salary range, work schedule
   - Time-based filtering (Latest, Last 24 hours, Last week, Last month)
   - Sort by salary (High to Low, Low to High) or date
   - Skill-based filtering with tags
   - Real-time job count display
   - Job details with company information, location, salary, tags

2. THREE-COLUMN LAYOUT
   - Left Sidebar: Navigation menu with Jobs, Resume, Profile, Agent, Coaching
   - Center Content: Main job listings with filters and search
   - Right Sidebar: AI Copilot assistant (Orion) for job recommendations

3. RESPONSIVE DESIGN
   - Desktop: Full three-column layout with sidebars
   - Tablet: Sidebars hidden, full-width main content
   - Mobile: Optimized single-column layout with bottom navigation
   - Touch-friendly interfaces (44px+ touch targets)
   - Smooth animations and transitions
   - Mobile filter drawer for easy access

4. AI-POWERED FEATURES
   - AI Copilot (Orion) for personalized job recommendations
   - "Ask Orion" functionality on job cards
   - AI-powered job matching and suggestions
   - Interactive chat interface with AI assistant

5. USER FEATURES
   - User authentication (Login/SignUp)
   - User profiles
   - Job application tracking (Liked, Applied, External tabs)
   - Messages/Notifications
   - Community section
   - Post jobs functionality
   - Admin dashboard

6. JOB POSTING
   - Post new job listings through UI
   - Automatic job data extraction from URLs
   - Integration with Supabase database
   - Job validation and error handling

7. EXTERNAL API INTEGRATION
   - RapidAPI integration for job listings
   - Adzuna API integration
   - Real-time job data synchronization
   - Background job fetching and updates

================================================================================
                        DATABASE SCHEMA
================================================================================

JOBS TABLE:
  - id (Primary Key)
  - company (Company name)
  - title (Job title)
  - location (Job location)
  - salary (Salary information)
  - company_logo (Logo URL)
  - tags (Array of skill tags)
  - link (Job posting URL)
  - description (Job description)
  - work_location (Remote/Onsite)
  - experience (Experience level)
  - working_schedule (Full-time/Part-time)
  - employment_type (Employment type)
  - posted_by (User who posted)
  - created_at (Timestamp)
  - updated_at (Timestamp)

MESSAGES TABLE:
  - id (Primary Key)
  - conversation_id (Conversation identifier)
  - sender_id (Sender user ID)
  - receiver_id (Receiver user ID)
  - message_text (Message content)
  - is_read (Read status)
  - created_at (Timestamp)

================================================================================
                        DEVELOPMENT WORKFLOW
================================================================================

LOCAL DEVELOPMENT:
  1. Install dependencies: npm install
  2. Start backend: cd backend && npm start (Port 5000)
  3. Start frontend: npm start (Port 3000)
  4. Access application: http://localhost:3000

TESTING:
  - Run tests: npm test
  - Run specific test: npm test -- ComponentName.test.js
  - Test API: node scripts/test-api.js
  - Test Supabase: node scripts/test-supabase.js

BUILDING:
  - Production build: npm run build
  - Output: build/ directory with optimized files

DEPLOYMENT:
  - Frontend: Deploy to Vercel
  - Backend: Deploy Express server
  - Database: Supabase cloud instance

================================================================================
                        API ENDPOINTS
================================================================================

FRONTEND API (src/lib/api.js):
  - GET /api/jobs - Fetch all jobs
  - POST /api/jobs - Create new job
  - GET /api/jobs/:id - Get job by ID
  - PUT /api/jobs/:id - Update job
  - DELETE /api/jobs/:id - Delete job

BACKEND ROUTES (backend/server.js):
  - GET /api/health - Health check
  - GET /api/jobs - Fetch jobs with filters
  - POST /api/jobs - Create job posting
  - GET /api/jobs/rapidapi - Fetch from RapidAPI
  - GET /api/jobs/adzuna - Fetch from Adzuna API
