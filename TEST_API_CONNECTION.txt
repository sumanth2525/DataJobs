TROUBLESHOOTING
================================================================================

Issue: Connection Refused
Solution:
  1. Make sure backend server is running
  2. Check if port 5000 is available
  3. Verify .env file exists in backend/ folder

Issue: 404 Not Found
Solution:
  1. Check if route exists in server.js
  2. Verify URL path is correct
  3. Check server logs for errors

Issue: 500 Internal Server Error
Solution:
  1. Check Supabase connection
  2. Verify environment variables
  3. Check server console for error details

Issue: CORS Errors
Solution:
  1. Backend has CORS enabled by default
  2. If issues persist, check backend/server.js CORS config
