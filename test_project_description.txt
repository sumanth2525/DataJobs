OVERVIEW
================================================================================

Data Job Portal is a modern, full-stack web application specifically designed
for data-related job roles. It serves as a comprehensive platform connecting
job seekers in data science, data analytics, data engineering, machine
learning, and related fields with employers looking to fill these positions.

The platform provides an intuitive interface for job searching, filtering,
and application, while also offering companies an easy way to post job
openings. Built with React for the frontend, Node.js/Express for the backend,
and Supabase for database management, it offers a scalable and maintainable
solution for the data job market.

================================================================================
KEY FEATURES
================================================================================

1. JOB SEARCH & FILTERING
   - Advanced search by job title, company, or keywords
   - Filter by work location (Remote, On-site, Hybrid)
   - Filter by experience level (Junior, Middle, Senior)
   - Filter by salary range (with min/max sliders)
   - Filter by working schedule (Full time, Part time, Internship, etc.)
   - Filter by employment type (Full day, Flexible, Shift work, etc.)
   - Time-based filtering (Latest, Last 24 hours, Last week, Last month)
   - Sort by salary (High to Low, Low to High)
   - Real-time filter application

2. COMPANY JOB POSTING
   - Easy job posting interface for companies
   - Automatic job information extraction from URLs
   - Support for multiple job board formats (LinkedIn, Indeed, Glassdoor, etc.)
   - QR code generation for easy job posting link sharing
   - Automatic parsing of job details from posted URLs

3. USER AUTHENTICATION
   - Secure login/signup with email or phone number
   - JWT-based authentication
   - Session management
   - Password hashing with bcryptjs
   - User profile management

4. LIVE FEATURES
   - Real-time online users count
   - Live messaging between users
   - Community section for connections and job sharing
   - Real-time updates and notifications

5. USER PROFILES
   - Complete user profile management
   - Contact information
   - Recruiter contact details
   - Profile customization options

6. RESPONSIVE DESIGN
   - Desktop web application experience
   - Mobile app-like interface on mobile devices
   - Progressive Web App (PWA) support
   - Installable on mobile devices
   - Touch-optimized interactions

7. EXTERNAL API INTEGRATION
   - RapidAPI integration for job search
   - Internships API for data-related roles
   - Real-time job data fetching
   - Multiple job data sources

================================================================================
INSTALLATION & SETUP
================================================================================

PREREQUISITES:
--------------
- Node.js (v14 or higher)
- npm or yarn
- Supabase account
- RapidAPI account (optional)

INSTALLATION STEPS:
-------------------
1. Clone repository: git clone https://github.com/sumanth2525/DataJobs.git
2. Install frontend dependencies: npm install
3. Install backend dependencies: cd backend && npm install
4. Set up environment variables (.env files)
5. Configure Supabase database
6. Start backend: cd backend && npm run dev
7. Start frontend: npm start

ENVIRONMENT VARIABLES:
----------------------
Frontend (.env):
- REACT_APP_API_URL
- REACT_APP_SUPABASE_URL
- REACT_APP_SUPABASE_ANON_KEY
- REACT_APP_RAPIDAPI_KEY
- REACT_APP_RAPIDAPI_APP

Backend (backend/.env):
- PORT
- SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY
- RAPIDAPI_KEY
- JWT_SECRET
- NODE_ENV
